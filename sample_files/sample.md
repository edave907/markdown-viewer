# Markdown Viewer

A simple Python-based markdown viewer with GitHub-like styling.

## Features

- **Bold text** and *italic text*
- `inline code` blocks
- Lists and nested items
- Code blocks with syntax highlighting

### Code Example

```python
def hello_world():
    print("Hello, World!")
```

### Lists

- First item
- Second item
  - Nested item
  - Another nested item
- Third item

### Blockquotes

> This is a blockquote example
> It can span multiple lines

---

## Installation

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

```bash
python markdown_viewer.py sample.md
```

Enjoy your markdown viewer!
